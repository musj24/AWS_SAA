# 용어
--------------------------------------
1. VPC : 사용자 전용 가상 네트워크, Region 단위
2. 서브넷 : VPC를 나눈 네트워크, AZ 단위, EC2가 배치되는 공간
   - 퍼블릭 : 인터넷으로 향하는 Route가 InternetGateway로 잡혀있음 (인터넷과 쌍방 통신 가능)
   - 프라이빗 : 인터넷으로 향하는 Route가 NAT gateway, 내부 -> 외부는 가능하지만 외부에서 내부 접근은 불가능

**
3. AWS Client VPN : remote vpn
4. AWS site to site VPN
5. AWS private link : VPC 에서 다른 VPC나 서비스에 인터넷 없이 연결가능 (특정 서비스에 연결에만 특화)
6. AWS direct connect : 다른 네트워크 (회사 등)과 AWS를 전용망으로 연
